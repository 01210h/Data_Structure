<h3>Algorithm arrayMax(A,n)</h3>
     <li>input array A of n integers</li>
     <li>output maximum element of A</li>
     <li>currentMax<-A[0]</li>
     <li>for i<-1 to n-1 do</li>
       <pre> if A[i]>currentMax then
              currentMax<-A[i]
            return currentMax
